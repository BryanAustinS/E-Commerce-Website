Notes:
When creating new API Endpoint
--Backend
1. Create model in model.py (create the DB model)
2. Create serializer in serializer.py (convert the DB result into JSON so that it can be used in JS)
3. Create View in views.py 
4. Add the URL in url.py
-- Frontend
5. Create new Vue page that outputs the page (create the HTML template and JS Script)
6. Add the page into router/index.js
